# Welcome to Gota Media Fluffy UI library ü¶Ñ
Fluffy is a opinionated but highly customizable UI library used by Gota Media AB.

## Installation
Install with NPM

<Source
    language={'bash'}
    code={`
        npm install @gotamedia/fluffy
    `}
/>

### Peer dependencies
Fluffy requires that you install following depdencies in your own project

### ESLint
Fluffy also shiped with some ESLint config to help you write awesome code ü¶Ñ 

You can simply extend your config with the Fluffy one
<Source
    language={'json'}
    code={`
{
    "extends": [
        "./node_modules/@gotamedia/fluffy/dev/eslint-config"
    ]
}
`}
/>

## Usage
Naming import from root

<Source
    language={'tsx'}
    code={`
import { Icon, Icons } from '@gotamedia/fluffy'
\nconst MyComponent = () => (
    <div>
        <Icon icon={Icons.RocketLaunch}/>
    </div>
)
    `}
/>

Direct import

<Source
    language={'tsx'}
    code={`
import Icon, { Icons } from '@gotamedia/fluffy/Icon'
\nconst MyComponent = () => (
    <div>
        <Icon icon={Icons.RocketLaunch}/>
    </div>
)
    `}
/>

## Theming
Import Fluffy's Theme to enable style for all Fluffy components.

<Source
    language={'tsx'}
    code={`
import { render } from 'react-dom'
import { ThemeProvider } from 'styled-components'
import { Button, getTheme } from '@gotamedia/fluffy'
\nconst App = () => (
    <div>
        <Button>
            {'It is so themeable! üë©‚Äçüé§'}
        </Button>
    </div>
)
\nconst theme = merge(
    getTheme(),
    {
        colors: {
            brand: '#E62267'
        }
    }
)
\nrender((
    <ThemeProvider theme={theme}>
        <App />
    </ThemeProvider>
), document.getElementById('root'))
    `}
/>
