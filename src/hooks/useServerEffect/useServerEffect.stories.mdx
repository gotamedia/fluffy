## About
useServerEffect is a custom useEffect hook that ONLY runs on server during SSR phase

## Usage
<Source
    language={'tsx'}
    code={`
import { useServerEffect } from '@gotamedia/fluffy'
\nconst Component = () => {
    useServerEffect(() => {
        // Do stuff...
    }, [])
`}
/>
