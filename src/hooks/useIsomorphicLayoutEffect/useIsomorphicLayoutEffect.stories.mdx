## About
useIsomorphicLayoutEffect is a helper hook for scheduling a layout effect with a fallback to a regular effect for environments where layout effects should not be used (such as server-side rendering)

## Usage

<Source
    language={'tsx'}
    code={`
import { useIsomorphicLayoutEffect } from '@gotamedia/fluffy'\n
const Component = () => {
    {...}\n
    useIsomorphicLayoutEffect(() => {
        {...}
    }, [])\n
    {...}\n
    return (
        <div>
            {...}
        </div>
    )
}
`}
/>
