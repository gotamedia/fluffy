## About
useMeasure is a helper hook that tracks dimensions of an HTML element using the Resize Observer API.

## Usage

<Source
    language={'tsx'}
    code={`
import { useMeasure } from '@gotamedia/fluffy'
\nconst Component = () => {
    const [element, setElementRef] = useState()
    \nconst elementRect = useMeasure(element)
    \nreturn (
        <div>
            <div ref={setElementRef}>
                {...}
            </div>\n
            {JSON.stringify(elementRect)}
        </div>
    )
}
`}
/>
