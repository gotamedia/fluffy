## About
React implementation of the Intersection Observer API to tell you when an element enters or leaves the viewport. Contains both a Hooks, render props and plain children implementation.

For more info please refer to [react-intersection-observer official DOCs](https://www.npmjs.com/package/react-intersection-observer)

## Usage

<Source
    language={'tsx'}
    code={`
import { ReactIntersectionObserver } from '@gotamedia/fluffy'
\nconst Component = () => {
    const { ref, inView } = ReactIntersectionObserver.useInView()\n
    return (
        <div ref={ref}>
            <h2>{'Header inside viewport: '}{inView}</h2>
        </div>
    )
}
`}
/>
